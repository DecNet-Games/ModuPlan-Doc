# Installation Guide

ModuPlan Pro is distributed safely and exclusively through the **Unity Asset Store**.

## How to Install ModuPlan Pro

1. Go to the **Unity Asset Store** page for ModuPlan Pro and complete your purchase.
2. Open your Unity Project (must be 2021.3+ with UIElements and Input System enabled).
3. Open the **Package Manager** (`Window > Package Manager`).
4. Select **My Assets** from the dropdown.
5. Search for `ModuPlan Pro`.
6. Click **Download**, then click **Import** when finished.
7. Ensure all assets are checked in the import window and proceed.

> **Caution:** ModuPlan Pro is a premium asset and is NOT available for free on GitHub. Please ensure you are logged into your Unity ID associated with the purchase.

## Dependencies

ModuPlan Pro relies on the **New Input System** and **UI Toolkit**. 
If you haven't enabled the new Input System:
- Go to `Edit > Project Settings > Player > Other Settings`
- Change **Active Input Handling** to **Both** or **Input System Package (New)**.
- Restart the Unity Editor when prompted.
